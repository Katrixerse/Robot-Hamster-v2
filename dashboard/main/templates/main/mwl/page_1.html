Placeholders
If you want to spice up your welcome or leave messages, you can use placeholders. To put it simply, placeholders are bits of text that will get replaced with specific information when a user joins or leaves.
So as an example, if you were to set the output to: Hey %NAME%, thanks for using this command in %CHANNELNAME%, channel of %GUILDNAME%.
the bot would convert that into: Hey {{username}}, thanks for using this command in #general, channel of Robot Hamster.
