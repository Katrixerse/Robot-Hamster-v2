Server Captcha
Server captcha is a feature that makes it so when a new user joins, they have to solve a captcha first. A captcha is a distorted image with some text in it, text which the new member has to enter in order to be able to talk in your server. Why? Using this feature is extremely effective against raids, since it is very hard for bots to detect the text in the image.
In the image above, a new user just joined. The bot sends a DM with the captcha.

What is Server Password?
{% bs_icon 'exclamation-triangle-fill' size='2em' color='#9ca6aa' %} Enabling this feature will disable Server Captcha, as they can't both work at the same time
In case you don't like captchas, you can set your own password that a user has to enter before being able to talk in your server (they won't be able to see the password). This can be used to prevent uninvited users in your guild, in case your invite gets leaked.

In the image above, a new user just joined. The bot sends a message informing them about the password.
